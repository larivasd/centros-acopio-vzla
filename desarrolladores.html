Datos abiertos del directorio de centros de acopio para los afectados por los terremotos del 24/06/2026. Úsalos libremente en tu app, bot o sitio.

Endpoint

GET https://centros-acopio-vzla.com/data/centros.json
Gratis Sin API key CORS abierto JSON Se actualiza 3× al día

Una sola petición GET devuelve todo el listado de centros con su información completa. No requiere autenticación. Se puede llamar desde el navegador (CORS habilitado).

Forma de la respuesta

{
  "actualizado": "2026-06-29",
  "fuente": "Directorios ciudadanos y medios ...",
  "centros": [
    {
      "pais": "Venezuela",
      "org": "Comando Con Venezuela",
      "estado": "Barinas",
      "ciudad": "Barinas",
      "municipio": "Barinas",
      "dir": "Av. Marqués del Pumar, diagonal al Hotel Comercio, Casa Azul",
      "reciben": "Agua, alimentos no perecederos, insumos médicos, ropa y abrigos",
      "tel": "",
      "info": "",
      "lat": 8.625737,
      "lng": -70.217709
    }
  ]
}

Campos de cada centro

CampoTipoDescripción
paisstringPaís del centro (Venezuela o el país de la diáspora).
orgstringOrganización o nombre del centro.
estadostringEstado / provincia / región.
ciudadstringCiudad.
municipiostringMunicipio (puede ir vacío).
dirstringDirección y referencias.
recibenstringQué donaciones reciben (puede ir vacío).
telstringTeléfono de contacto (puede ir vacío).
infostringEnlace (Instagram/web) u horario (puede ir vacío).
latnumber\|nullLatitud (GPS).
lngnumber\|nullLongitud (GPS).

Ejemplos

cURL

curl https://centros-acopio-vzla.com/data/centros.json

JavaScript (fetch)

const res = await fetch('https://centros-acopio-vzla.com/data/centros.json');
const { centros } = await res.json();
console.log(centros.length + ' centros');

// ej: solo los de Venezuela con coordenadas
const conMapa = centros.filter(c => c.pais === 'Venezuela' && c.lat != null);

Python

import requests

data = requests.get('https://centros-acopio-vzla.com/data/centros.json').json()
print(len(data['centros']), 'centros')

for c in data['centros']:
    print(c['org'], '—', c['ciudad'], c['pais'])

Notas de uso

  • Los datos se recopilan de fuentes públicas y se actualizan automáticamente 3 veces al día (8:00, 12:00 y 18:00, hora de Venezuela).
  • Algunas coordenadas son aproximadas (a nivel de barrio o ciudad) cuando no hay dirección exacta.
  • La respuesta tiene caché de ~10 minutos.
  • Verifica siempre horario y necesidades antes de dirigir a alguien a un centro: la situación cambia rápido.
  • Uso libre. Si la integras en algo, agradecemos que enlaces a centros-acopio-vzla.com.
